网站建立

本部落格是如何建立的。

科技

#
  • 使用静态网页生成器 Eleventy 版本 3.0.0 构建
  • JavaScript 运行环境：Bun。
  • 模板语言：Nunjucks 和 Markdown。
  • 使用 Visual Studio Code 编写代码。
  • 测试浏览器：Mozilla Firefox、Vivaldi 和 Lynx。
  • 源代码可在 GitHub 上获取。

托管与域名

#
  • 通过 GitHub action 从本部落格的 GitHub 仓库部署到 Hostinger。
  • 本部落格网址 blog.helenchong.omg.lol 是我的 omg.lol 网址 helenchong.omg.lol 的子域名。我于 2024 年 10 月 19 日注册并购买了 omg.lol 的网址与终身会员资格。

插件

#
  • Eleventy 的官方 RSS 插件
  • Eleventy 的官方 Fetch 插件
  • uncenter 的 eleventy-plugin-toc
  • limax

markdown-it 插件

#

Eleventy 内置了 Markdown 解析器 markdown-it 以将 Markdown 转换为 HTML。

  • markdown-it-anchor
  • markdown-it-attrs
  • markdown-it-bracketed-spans

字体

#
  • Lexica Ultralegible——拉丁字母和数字的默认字体。
  • Intel One Mono——单行距字体。

品牌

#
  • 我用 Affinity Designer 2 设计银河花园的标志。
  • 我用 Adobe Illustrator CS5 绘制自己的卡通插图头像，绘画风格晾干源自 Andy Carolan。
